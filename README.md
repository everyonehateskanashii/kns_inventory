# kns_inventory 1.2.0

Inventory HUD for ESX. You can open and close inventory using F2.

## Requirements
* [es_extended](https://github.com/esx-framework/es_extended/tree/v1-final) 

## Features
- Using items
- Dropping items
- Giving items
- Cash included
- Accounts support (bank, black money, ...)
- Weapons support
